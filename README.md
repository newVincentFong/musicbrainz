## Development

### Environment

```bash
node -v
# v18.16.1 (or above)
pnpm -v
# v8.9.0 (or above)
```

### Setup

```bash
pnpm install
```

### Run

```bash
pnpm run dev
# Web
pnpm run start
# Electron
```
